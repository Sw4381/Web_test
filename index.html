Security Intelligence Lab

  • 호서대 보안지능연구실에서는 최첨단 AI 기술과 심도 있는 데이터 분석을 융합하여 다양하고 복잡한 보안 위협에 맞서는 연구를 수행하고 있습니다.
  • 데이터 마이닝, 기계학습, 심층학습 등의 최신 기술을 활용하여 현장에서 직면하는 보안 문제에 대한 실용적이며 혁신적인 솔루션 개발에 주력하고 있습니다.
  • 다양한 정부 과제에 참여하며 국내 주요 기업들과의 협약을 통해 실무적 적용 가능성을 높이는 연구를 진행하고 있습니다.

About SILab

  • AI 기술연구 및 보안분야에 응용을 기본으로 하며,실질적인 AI 생산성을 높이기 위해 AI decision support(XAI), AI 오탐지 대상 식별/조치(false alarm detection),
  • AI 모델의 update 시점 분석(drift detection), adversarial attack에 대한 AI 강건성 수준분석(AI robustness),Graph 기반 network attack, threat intelligence 분석 등
  • 의 연구를 진행중에 있습니다.

Detailed research

  • [R10] Graph Based Threat Intelligence - Graph 기반 다수 Entity(Group/Technique/Software/Domain..) 간의 관계 분석
  • [R9] Network Attack Detection - 암호화된 트래픽 환경에서의 attack IP 탐지, topology 패턴 분석 기술
  • [R8] AI Model Drift Detection - AI 모델의 갱신 시점 자동탐색 기술
  • [R7] Reference Based Explanation - Secure 분야의 효과적 AI 예측에 대한 Reference 기반 해석 기술
  • [R6] XAI Technique Evaluation - 다양한 XAI 기술에 대한 범용적인 성능평가 Framework
  • [R5] Robust AI Model Development - Adversarial attack에 대응 가능한 Robust한 AI Model 개발 및 Robustness 수준측정 기술
  • [R4] Illegal Content Detection - 불법 콘텐츠 탐지 자동 탐지 및 저작권 침해자 프로파일링 기술
  • [R3] False Alarm Detection - Security 시스템에서의 False alarm 선별 기술
  • [R2] Unknown Endpoint Attack Detection - EDR에서의 unknown attack 탐지 기술
  • [R1] Predictive Maintenance - Smart factory에서의 공작기계 고장시점을 예측하고, 고장원인에 대한 분석 기술
