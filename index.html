Security Intelligence Lab

보안 지능 연구실에서는 AI 기술 연구 및 보안분야에 응용을 기본으로 하며

실질적인 AI 생산성을 높이기 위해 아래와 같은 연구를 진행하고 있습니다.

주요 연구

[R10] Graph Based Threat Intelligence

- Graph 기반 다수 Entity(Group/Technique/Software/Domain..) 간의 관계 분석

- 복잡한 node 관계에서의 다양한 해석 지원

[R9] Network Attack Detection

- 암호화된 트래픽 환경에서의 attack IP 탐지, topology 패턴 분석 기술

- 암호화된 환경에서 공격자 탐색 기반 마련

[R8] AI Model Drift Detection

- AI 모델의 갱신 시점 자동탐색 기술

- Supervised, Unsupervised 환경에서의 최적화된 모델 운영 지원

[R7] Reference Based Explanation

- Secure 분야의 효과적 AI 예측에 대한 Reference 기반 해석 기술

- 직관적인 해석 제공에 따라 AI-Decision support

[R6] XAI Technique Evaluation

- 다양한 XAI 기술에 대한 범용적인 성능평가 Framework

- XAI 성공 평가를 통한, XAI 신뢰성 확보

[R5] Robust AI Model Development

- Adversarial attack에 대응 가능한 Robust한 AI Model 개발 및 Robustness 수준측정 기술

- Adversarial attack의 취약한 공격유형 식별 및 사전대응 지원

[R4] Illegal Content Detection

- 불법 콘텐츠 탐지 자동 탐지 및 저작권 침해자 프로파일링 기술

- 불법 복제작물 실시간 유포자 추적/검거 지원

[R3] False Alarm Detection

- Security 시스템에서의 False alarm 선별 기술

- False alarm 제거를 통한 효율적인 분석인력 운영 지원

[R2] Unknown Endpoint Attack Detection

- EDR에서의 unknown attack 탐지 기술

- Unknown attack에 대한 threat hunting 지원

[R1] Predictive Maintenance

- Smart factory에서의 공작기계 고장시점을 예측하고, 고장원인에 대한 분석 기술

- 비용대비 효과적인 부품교체 시기 결정 지원
